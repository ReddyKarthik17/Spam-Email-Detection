# Spam Email Detection using Machine Learning

## Project Overview
This project detects whether a message is spam or not spam using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Features
- Text preprocessing
- Spam classification
- Machine learning model training
- Message prediction

## How to Run

Install libraries:

```bash
pip install -r requirements.txt
```

Run project:

```bash
python spam_detector.py
```

## Output
The model predicts whether a message is spam or ham (not spam).
